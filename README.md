# VRC / HRV RRi Analyzer v3.8

Versión corregida para comparación de varios registros RRi.

Incluye:

- Basal + E1-E6 + R1-R3.
- Ventanas de duración libre.
- Comparación entre dos o más registros.
- Selección de fases a comparar.
- Selección de variable principal.
- Gráfica comparativa entre registros.
- RRi superpuesto dentro de una fase.
- Exportación Excel y ZIP.

Ejecutar:

```bash
streamlit run app.py
```
